Yazılım geliştirme süreçlerinde olmazsa olmazlardan olan dağıtık versiyon kontrol sistemlerinden belkide en çok kullanılan git'i öğrenmek için kısa bir süredir çaba sarf ediyorum.   • Git Püf Noktaları -> http://vigo.github.io/git-tips/
  • Son olarakta çok aktif olmasada bu twitter adresinide takip edebilirsiniz -> https://twitter.com/GiT_Tips_TR

 

Mutlaka uygulama yaparak öğrenmeye çalışmalısınız örneğin bende daha yeni yeni öğrenmeye başlamışken sitemin temasını github üzerinde git kullanarak yönetmeye çalışıyorum. Zamanla deneme-yanılma ile öğrenmeyi hedefliyorum. Git iyidir, git güzeldir :) .
